Situs Web

Situs web adalah seranhkaian halaman web yang memiliki informasi didalamnya

Situs web bersifat public, namun dalam praktiknya tidak semuanya public

Situs web dapat diakses melalui jaringan internet atau intranet dengan menggunakan web browser

Situs web terdiri dari halaman web yang saling terhubung satu sama lain dengan menggunakan hyperlink

Situs web dapat berisi berbagai jenis konten, seperti teks, gambar, video, audio, dan elemen interaktif lainnya

keseimbangan visual

Kombinasi elemen-elemen visual pada halaman web harus seimbang, sehingga tidak ada bagian yang terlalu berat atau terlalu ringan

Kebanyakan situs web menggunakan keseimbangan simetris, di mana elemen-elemen di sisi kiri dan kanan halaman web memiliki bobot visual yang sama

pengulagan elemen-elemen visual pada halaman web dapat menciptakan keseimbangan yang harmonis

prinsip selanjutnya adalah kontras, di mana elemen-elemen yang berbeda memiliki perbedaan yang mencolok, sehingga dapat menarik perhatian pengguna
